Random Forest#

An ensemble of decision trees used for classification and regression tasks.

  • Typically trained using the “bagging” method:

    • Trained on randomly selected subsets of the training data.

  • The algorithm constructs multiple decision trees and combines them to produce a more accurate and stable prediction, introducing additional randomness while growing the trees.

  • Final Decision:

    • Soft voting is used for classification.

    • Averaging is used for regression.

Random Forest Classifier#

from sklearn.datasets import load_breast_cancer
X, y = load_breast_cancer(return_X_y=True)

from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(X, y, random_state=0)

from sklearn.ensemble import RandomForestClassifier
rf = RandomForestClassifier(random_state=0)

rf.fit(X_train, y_train)

n_estimators#

This parameter is used to specify the number of decision trees to be included in the model.
